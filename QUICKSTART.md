# DevOps Security Monitor - Quick Start

## Installation

```bash
# Clone the repository
git clone https://github.com/alexkore12/devops-security-monitor.git
cd devops-security-monitor

# Run setup
./setup.sh

# Start monitoring
./monitor.sh
```

## Usage

- `./monitor.sh` - Run security monitoring
- `./alerts.sh` - Configure alert notifications
- `./security_scanner.sh` - Run vulnerability scanning
